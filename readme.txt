This plugin manage your single hotel and booking.

== Description ==

Hotel booking system wordpress plugin is a dedicated booking platform for hotel room booking. This plugin is designed for managers of a hotel for performing complete administrative tasks and guests’ public booking. The backend admin section gives opportunity to manage rooms, amenities, custom pricing, booking, payments in a flawless manner. Front­end section of this plugin provides facility to guests to apply for reservation with online payment option flexibility.

== Features ==

* Booking management
* Reservation management
* Room availability calendar
* Room management
* Amenity management
* Floor management
* Additional service management
* Custom pricing for room
* Online payment with paypal and stripe
* Multisite / multihotel booking
* Booking filter for active, reservation & completed
* Payment report

== Installation ==

1. Upload the plugin zip file to the `/wp-content/plugins/` directory and unzip it, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to Settings screen of the plugin and configure the plugin according to your need.
4. Go to Chambers screen of the plugin and configure your schedule.
5. Now you can go to the Appointment screen of the plugin and start discovering it's features.
